# GLPI-TacticalRmmPlugin

Integration between GLPI and TacticalRMM for asset management.

## Features
- Configure TacticalRMM URL and relation field
- Simple configuration form

## Requirements
- GLPI 11.x
- PHP 8.4+

## Installation
1. Copy this plugin to your GLPI `plugins/` directory.
2. Install and activate via GLPI web interface.
